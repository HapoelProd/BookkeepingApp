# הפועל ירושלים כדורסל - מערכת הנהלת חשבונות

## אפליקציית ווב מקצועית לעיבוד נתוני הנהלת חשבונות

### תיאור הפרויקט
אפליקציית ווב מקצועית שהופכת את הסקריפט Python שלך לפלטפורמה ידידותית למשתמש עם:
- העלאת קבצי CSV בגרירה ושחרור
- עיבוד אוטומטי של הנתונים
- תצוגת תוצאות באקרטיות
- הורדת קובץ Excel מעובד
- בדיקת איזון אוטומטית

### הוראות הפעלה מהירות

#### 1. התקן Python packages
```bash
pip install -r requirements.txt
```

#### 2. הפעל את האפליקציה
```bash
python web_app.py
```

#### 3. פתח בדפדפן
```
http://localhost:5000
```

### מבנה התיקיות
```
BookkeepingApp/
├── web_app.py              # האפליקציה הראשית
├── app.ipynb              # הסקריפט המקורי שלך (נשמר)
├── requirements.txt        # תלויות Python
├── README.md              # קובץ זה
├── static/
│   ├── css/style.css      # עיצוב האתר
│   ├── js/main.js         # פונקציונליות JavaScript
│   └── uploads/           # תיקיה זמנית לקבצים
└── templates/
    ├── base.html          # תבנית בסיס
    ├── upload.html        # דף העלאה
    └── results.html       # דף תוצאות
```

### תכונות האפליקציה

####  דף העלאה
- כפתור העלאה גדול וברור
- גרירה ושחרור של קבצים
- אימות קובץ אוטומטי (CSV בלבד, עד 16MB)
- הנחיות שימוש ברורות

####  דף תוצאות
- בדיקת איזון צבעונית
- טאבים לכל גיליון:
  - ללא פרסומים (without_ad)
  - פרסומים (advertisement)
  - נתונים נוספים (rest)
  - סיכום (Summary)
- הורדת קובץ Excel

####  עיצוב
- צבעי הפועל ירושלים: שחור, לבן, אדום (#D50000)
- ממשק נקי ומינימליסטי
- תמיכה בעברית מלאה
- רספונסיבי לכל הזיזבלים

### הסבר טכני

#### המחלקות העיקריות
- `BookkeepingProcessor`: מכילה את כל הלוגיקה המקורית שלך
- `Flask App`: מנהל את הדרכים והממשק
- `Templates`: עמודי HTML דינמיים
- `Static Assets`: CSS, JS ותמונות

#### תהליך העבודה
1. משתמש מעלה קובץ CSV
2. האפליקציה שומרת זמנית ומעבדת
3. הלוגיקה המקורית שלך רצה בדיוק
4. תוצאות מוצגות בטאבים
5. קובץ Excel זמין להורדה

### שגיאות נפוצות ופתרונות

#### "Module not found"
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

#### "Port already in use"
עדכן את השורה האחרונה ב-`web_app.py`:
```python
app.run(debug=True, host='localhost', port=5001)
```

#### "Permission denied"
```bash
chmod 755 static/uploads
```

#### טקסט עברי לא מוצג
וודא שקובץ ה-CSV מקודד ב-UTF-8

### הפעלת הייצור (Production)

#### עם Gunicorn
```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 web_app:app
```

#### עם Docker
```dockerfile
FROM python:3.9-slim
COPY . /app
WORKDIR /app
RUN pip install -r requirements.txt
CMD ["python", "web_app.py"]
```
